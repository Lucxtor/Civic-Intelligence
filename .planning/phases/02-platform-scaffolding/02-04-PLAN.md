<task id="02-04-B">
<action>
Create `src/components/admin/DemographicManager.tsx`. This component:
1. Fetches all `demographic_dimensions` from Supabase on mount.
2. Displays each dimension as a `glass-panel` card showing: Label, Key (badge), and option pills.
3. Has an "Add Dimension" button that shows an inline form with:
   - `Label` text input (e.g., "Occupation")
   - `Key` text input (auto-slugified from label, e.g., "occupation") — read-only after auto-fill.
   - Dynamic options list: text inputs you can add/remove per option, with an "Add Option" button.
   - "Save Dimension" button — calls `supabase.from('demographic_dimensions').insert(...)`.
4. Each existing dimension card has a "Delete" icon button that calls `supabase.from('demographic_dimensions').delete().eq('id', id)`.
5. On data mutations, re-fetches the full list.

UI conventions: use `glass-panel` for cards, `border-border/50` for separators, `text-ipe-green` for add actions, `text-red-400` for delete actions.
</action>
